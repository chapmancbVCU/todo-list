Class: TodoItem

TodoItem(itemType, parentProject, title, description, dueDate, priority, isComplete;)

The TodoItem class is responsible for creating todo item objects. It extends the DataHandler class which supports CRUD operations.

Constructor

new TodoItem(itemType, parentProject, title, description, dueDate, priority, isComplete;)

Constructor for TodoItem.
Parameters:
Name Type Description
itemType String The type of item. It can be a todo list item, project, or note.
parentProject String The parent project for this particular tod list item.
title String The title for this todo list item.
description String Information that is particular to this todo list item.
dueDate String The date that this item must be completed.
priority String This value can be low-priority, medium-priority or high-priority. This suggests the urgency for this particular todo list item.
isComplete; Boolean
